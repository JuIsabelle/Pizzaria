<p align="left">🍕 BELLA MASSA<br>Este é um projeto de um site institucional e interativo para uma pizzaria, desenvolvido para colocar em prática conceitos fundamentais de desenvolvimento web front-end.<br><br>🚀 Funcionalidades<br>Menu Digital: Listagem de pizzas com preços e ingredientes.<br><br>Carrinho de Compras: Adição e remoção de itens com cálculo de total em tempo real.<br><br>Design Responsivo: Otimizado para visualização em celulares, tablets e desktops.<br><br>Validação de Endereço: Campos obrigatórios para entrega com verificação simples para evitar envios sem informações básicas.<br>
